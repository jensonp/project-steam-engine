# Learn PostgreSQL Workflow

## Reading Documentation

1. **Open the PostgreSQL docs in a split view**:
   - Press `Cmd + \` to split your editor
   - Open the doc PDF or use the browser panel

2. **Recommended reading order**:
   - Tutorial: https://www.postgresql.org/docs/current/tutorial.html
   - Tables: https://www.postgresql.org/docs/current/ddl-basics.html
   - Queries: https://www.postgresql.org/docs/current/queries.html
   - Joins: https://www.postgresql.org/docs/current/queries-table-expressions.html#QUERIES-JOIN
   - Aggregates: https://www.postgresql.org/docs/current/functions-aggregate.html
   - Indexes: https://www.postgresql.org/docs/current/indexes.html

## Asking Questions

While reading, ask me:
- "Explain [concept] with an example for Steam games"
- "Show me how [SQL concept] works for collaborative filtering"
- "Create a practice exercise for [topic]"

## Practice Pattern

```
1. Read a section of the docs
2. Ask me to explain or create an exercise
3. Try it on your test database
4. Ask me to review or debug
```
